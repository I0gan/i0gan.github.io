C++中的补码公式与位域

补码公式

-x = x+1 = ~(x-1)
~x = -x-1
-(x) = x+1
~(-x) = x-1

x+y = x - y-1 = (x|y) + (x&y)
x-y = x + ~y+1 = (x|y) - (x&y)
x^y = (x|y) - (x&y)
x|y = (x& ~y) + y
x&y = (x|y) - ~x

x==y : (x-y|y-x)
x!=y : x-y|y-x
x<y : (x-y) ^ ((x^y) & ((x-y) ^x))
x<=y : (x|y) & ((x^y) | (y-x))
x<y : (x&y) | ((x|y) & (x-y)) //unsigned
x<=y : (x|y) & ((x^y) | ~(y-x)) //unsigned
