查找目录下的所有文件中是否含有某个字符串

find .|xargs grep -ri "SMS_ISSEND"

查找目录下的所有文件中是否含有某个字符串,并且只打印出文件名

find .|xargs grep -ri "SMS_ISSEND" -l
