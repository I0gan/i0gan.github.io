修改三处 2.6 源码文件

  • 1.arch/x86/vdso/Makefile 中第 28 行的 -m elf_x86_64 改成 -m64，第 72 行的-m elf_i386 改成-m32
  • 2.drivers/net/igbvf/igbvf.h 中注释第 128 行
