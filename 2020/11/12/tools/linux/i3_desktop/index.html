日常使用快捷键

一下是我配置的快捷键

i3默认的mod键为alt

alt+r : 打开终端

alt+. : 打开程序搜索

alt+e : 命令行打开文件管理器

alt+f4 : 退出窗口：

alt+shift+r : 重新载入i3

alt+shift+e: 关闭i3

alt+方向/jkl: 在两个窗口中移动

alt+num: 打开工作区

alt+shift+num: 将当前窗口移动到某工作区

如何实现分屏

先按$mod+a焦距于父窗口，在通过$mod+h或者$mod+v控制接下来新的窗口以什么方式进行创建。
